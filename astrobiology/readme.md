# Computational Astrobiology Project

## Belgrade University

##Author: Vicente Arévalo

## About the repository
This repository collects all the information related to the Computational Astrobiology Project

## Description
The project covers all the steps that I user needs to folow in order to simulate a star-black hole system with a planet orbiting in a S-Type orbit. Moreover, the main aim of the project is to examine whether a planet transiting the star could produce detectable signals and how the presence of the black hole might influence the system.
